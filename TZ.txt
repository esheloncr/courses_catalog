В рамках задачи необходимо разработать каталог курсов.
Курс должен иметь название, описание и дату начала и окончания. Каталог должен иметь следующие функции и возможности:
Добавление курса в каталог
Удаление курса
Поиск курса по атрибутам(название, описание, дата начала)
Изменение атрибутов курса
Детальная страница курса должна отоброжать полную информацию о курсе.
Соискатель должен разработать готовое решене, т.е. кроме внутренней логики должно быть внешнее представление. 
Для внешнего представления можно использовать любимый css фреймворк( например bootstrap).
Выполненное задание должно распологаться на GitHub или Bitbucket
Удачи!
